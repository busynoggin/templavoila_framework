================================
Info.yaml & Screenshot.jpg
================================

Info.yaml
================================

The Info.yaml file contains the title and description of your skin to display in the skin selector.

**Example:** ::

    title: Busy Noggin Wireframe Skin
    description: Skin to use as a wireframe and as a base for new skins

Screenshot.jpg
================================

The Screenshot.jpg is an image to display in the skin selector.
